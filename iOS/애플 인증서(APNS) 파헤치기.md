# 애플 인증서(APNS) 파헤치기
## 인증서작업
### 인증서란?
**인증서는 이 컴퓨터로 개발하고 배포하겠다고 주어지는 증명자격**
### 인증서 생성 절차
1. 키체인 접근
2. 메뉴 인증서 지원 - 인증기관에서 인증서 요청
3. 정보입력 - 서명할 이메일 주소
	* 사용자 이메일 주소 - 서명할 이메일 주소
	* 일반 이름 - 서명할 사람 이름
	* CA 이메일 주소 - 비워둔다
	* 디스크에 저장됨
	* 본인이 키값 직접 지정

4. 생성되는 CSR 파일은 무슨 일이 있어도 저장!!

## Identifiers 작업
1. Identifiers 에서 앱 생성
2. 생성된 identifers(앱)에서 push 인증서 만들어주기(CSR 필요)
